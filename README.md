# The Ringing Truth

Living with Tinnitus Without Losing Your Mind

## About

The Ringing Truth is an independent resource dedicated to helping people understand, manage, and make peace with tinnitus. We publish evidence-based articles weaving neuroscience, clinical research, and contemplative wisdom.

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

